# 🌐 Content Marketing Agency Landing Page

A modern, responsive, and animation-rich landing page built for a **content marketing agency**. This project focuses on creating a premium agency-style web experience with strong visual hierarchy, smooth scrolling, bold typography, interactive UI elements, and polished frontend animations.

The website is designed as a frontend landing page to showcase the brand identity, services, creative direction, and digital marketing expertise of a modern content marketing agency.

- **Live Site**: <a href="https://rise-at-seven-sage.vercel.app" target="_blank">https://rise-at-seven-sage.vercel.app</a>

---

## 🔗 Project Overview

This project is a frontend website inspired by a **search-first content marketing agency** landing page. It emphasizes clean design, smooth user experience, responsive layouts, and high-quality animations.

The main goal of this project is to demonstrate advanced frontend development skills by recreating a professional agency-style landing page using modern web technologies.

The website includes a polished user interface, animated sections, responsive navigation, smooth scrolling behavior, and interactive visual components that make the landing page feel dynamic, modern, and professional.
